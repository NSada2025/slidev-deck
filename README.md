# ChoiAca Slidev Presentation

学生の研究選択支援プラットフォーム「ChoiAca」のプレゼンテーション資料です。

## 🌐 オンライン版

https://nsada2025.github.io/slidev-deck/

## 🎮 操作方法

- **→** または **Space**: 次のスライドへ
- **←**: 前のスライドへ  
- **F**: フルスクリーンモード
- **O**: スライド一覧表示

## 💻 ローカル実行

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev
```

## 📄 PDF出力

```bash
npm run export
```
